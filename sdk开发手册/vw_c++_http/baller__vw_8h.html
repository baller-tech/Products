调用 baller vw sdk 的接口前需先调用 BallerLogin 进行登录和调用者身份的验证。
BallerLogin 接口调用成功后需调用 BallerVWSessionBegin 接口创建一个语音唤醒功能的session，语音唤醒操作都是基于session的，当不再需要语音唤醒功能后，需调用 BallerVWSessionEnd 接口回收该 session。

◆ BallerVWAbort()

BALLER_API int BallerVWAbort ( const baller_session_id  session_id)

终止已传入音频数据的唤醒词检测

将音频数据实时传入BallerVWPut接口时，如果已经发现唤醒词且不需要继续等待唤醒时需调用该接口， SDK会丢弃已传入BallerVWPut接口但未来得及处理的音频数据。调用完该接口之后，如果继续调用BallerVWPut接口，会重新开始唤醒。

参数
[in]session_id需要中断的 session 的 sessioin id。
返回
0表示成功；其他表示失败，详情见错误码。
注解
int ret = BallerVWAbort(session_id);
if (BALLER_SUCCESS != ret)
{
printf("BallerVWAbort failed. error code: %d\n", ret);
}

◆ BallerVWPut()

BALLER_API int BallerVWPut ( const baller_session_id  session_id,
const char *  params,
const void *  in_data,
const int  in_len 
)

向 session 中放置采样数据

参数
session_id处理音频采样数据的 session 的 session id。
params处理采样数据的参数，详见下表：
参数 参数介绍 是否必填 可设置的值 值介绍
input_mode 输入模式 否 once 【默认值】将所有音频数据一次性传入
continue 音频数据时实时传入且不是最后一次传入
end 音频数据实时传入且本次是最后一次传入
in_data采样数据。
in_len采样数据的长度；以字节单位。
返回
0表示成功；其他表示失败，详情见错误码。
备注
音频采样需按照BallerVWSessionBegin中设置的采样率和采样点大小进行；只需要单个通道的采样数据即可。
注解
char* audio_data = 0;
int audio_len = 0;
// 获取音频数据，将获取的音频数据保存到audio_data变量中，将获取的音频数据长度保存到audio_len中
const char* params = "input_mode=once";
int ret = BallerVWPut(session_id, params, (void*)audio_data, audio_len);
if (BALLER_SUCCESS != ret)
{
printf("BallerVWPut failed. error code: %d\n", ret);
}

开始一个语音唤醒sessioin

参数
[in]session_params启动session的参数，详见下表：
参数 参数介绍 是否必填 可设置的值 值介绍
res_dir 资源文件所在的路径 是 资源文件所在的路径 资源文件所在的路径
wakeup_word 唤醒词 是 使用英文分号分割不同的唤醒词；使用空格分割唤醒词中不同的字/单词
中文使用大写拼音表示；英文使用小写字母 		唤醒词
engine_type 引擎类型 否 local 【默认值】使用本地引擎
cloud 【暂不支持】使用云端引擎
hardware 用来计算的硬件信息；当engine_type=local时有效 否 cpu_slow 【默认值】在CPU上进行计算
cpu_fast 【暂不支持】在CPU上快速进行计算
gpu 【暂不支持】在GPU上进行计算
vad 是否启用端点检测功能 否 on 【默认值】开启端点检测功能
off 不开启端点检测功能
sample_rate 音频的采样率 否 16 【默认值】音频采用16KHz的采样率进行采样
8 【暂不支持】音频采用8KHz的采样率进行采样
sample_size 音频的每个采样点大小 否 16 【默认值】音频每个采样点16bit
8 【暂不支持】音频每个采样点8bit
[in]callback唤醒结果的通知回调
[in]user_param用户自定义参数，唤醒结果的通知回调中会携带该参数
[out]session_id本次启动的session id
返回
0表示成功；其他表示失败，详情见错误码。
注解
void BALLER_CALLBACK my_callback(void* user_param, const int id)
{
printf("The awakened keyword index is: %d\n", id)
}
baller_session_id session_id = BALLER_INVALID_SESSION_ID;
const char* session_prams = "res_dir=data,wakeup_word=ZHONG GUO;hello LI LEI;good morning,engine_type=local,hardware=cpu_slow,vad=on,sample_rate=16,sample_size=16";
int ret = BallerVWSessionBegin(session_prams, my_callback, 0, &session_id);
if (BALLER_SUCCESS != ret)
{
printf("BallerVWSessionBegin failed. error code: %d\n", ret);
}
